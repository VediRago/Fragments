# Project

This folder keeps the project-level rules and foundations for **Fragments**.

It is not the main reading route for prose.

Use this folder to check:

- canon clarity rules
- core story foundation
- prose progression rules
- deeper terminology that should stay consistent

## Current Files

- `00_Canon_Clarity_Rules.md` — rules for keeping canon clear and causal.
- `01_Project_Foundation.md` — merged project foundation, core concepts, title logic, and prose progression.

## Rule

Do not let this folder become a dumping ground.

If a note is about a specific act, character, prose section, or bridge, move it to the folder where it belongs.

If a note explains the whole project, merge it here only when it helps a new reader understand the structure more clearly.
