# Characters

Use this folder for character identity, face consistency, outfit tests, poses, and scene images.

## Current priority characters

- Kael
- Sera
- Erit

## Standard character folder layout

Each main character has:

- `00_Main_Reference` — strongest identity image.
- `01_Face_Eyes` — close-ups, eye color, face consistency.
- `02_Outfits` — clothing and design tests.
- `03_Poses` — body language and action references.
- `04_Scenes` — story panels involving the character.
- `99_Rejected_Old` — old versions to keep but not use.
