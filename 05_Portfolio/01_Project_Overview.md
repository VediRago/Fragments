# Project Overview

## Summary

Fragments is a long-form narrative project built around a central design challenge:

How can readers discover deep connections between characters without being directly told those connections exist?

The story follows multiple lives across different settings, relationships, and time periods. Rather than relying on exposition, the project uses recurring patterns, behaviors, imagery, and emotional continuity to gradually reveal hidden connections.

## Core Design Goals

- Prioritize experience before explanation.
- Allow readers to reach conclusions through evidence.
- Build meaning through recurring patterns rather than direct reveals.
- Use character relationships as the primary vehicle for large-scale narrative ideas.
- Preserve mystery without sacrificing emotional clarity.

## Scope

The repository contains:

- Story structure and act planning
- Character architecture
- Narrative design notes
- Selected prose
- Development documentation

The underlying design evolved through iterative problem solving, with many major discoveries emerging from attempts to connect existing scenes rather than adding new exposition.
