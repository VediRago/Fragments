This folder contains the current worldbuilding layer for Erit's world.

It is not meant to be read like a finished encyclopedia.

The files show how the world is being built from story pressure, emotional causes, historical consequences, and possible game-facing conflicts.

## Current Core Chain

The worldbuilding currently follows this chain:

> Sera sees statue  
> -> statue belongs to Viriatus  
> -> Viriatus reveals a buried displacement event  
> -> House Ventari turns him into weapon, myth, and public memory  
> -> public displeasure with the old order allows Ventari discipline to look like rescue  
> -> Viriatus's victories become proof  
> -> Viriatus's private murder becomes public noble death  
> -> Regulatus grows from the interpretation of his death  
> -> founding lines inherit different wounds  
> -> House Devaar eventually rises through medicine, preservation, augmentation, and bodily dependency  
> -> Erit's era becomes shaped by identity, access, treatment, law, and control

## Design Method

The main method used for this world is:

> experience -> emotion -> reason -> behavior

If the idea expands, it becomes:

> behavior -> atmosphere -> history

For quest thinking, the method becomes:

> world pressure -> character need -> obstacle -> approaches -> consequence

## Current Worldbuilding Rule

Build only what creates one of the following:

- story pressure
- character behavior
- social conflict
- player-facing choices
- quest consequences
- stronger emotional or historical causality

If a detail does not create pressure, behavior, consequence, or meaning, it should stay undeveloped until the story needs it.

## Current Focus

The active focus is:

- House Ventari founding era
- Viriatus
- Ecalus Ventari
- Ermerus, Rutare, Ilunis, and Teresta
- Regulatus
- ACD / Celcinus Calendar
- AVD beginning with the public declaration of Regulatus
- the founding lines that later create pressure for Erit's era

## Important Constraint

Displacement exists in the history of Erit's world, but it must remain rare and poorly understood.

If displacement became common or scientifically understood, the mystery around Sera, Kael, and Erit would weaken.

The historical founder can reinforce that displacement has happened before, but he cannot explain Unity or replace the core Fragments characters.
