# Fragments

Fragments is a narrative design project exploring reader-driven discovery, emotional continuity, and hidden structural connections across multiple lives.

This repository contains portfolio material, story structure, character notes, prose samples, and deeper reference documentation.

## Repository Structure

- `05_Portfolio/` — recruiter-facing overview, case studies, prose index, and development process
- `04_Prose/` — selected prose scenes and excerpts
- `03_Acts/` — act structure and story progression
- `02_Characters/` — character reference files
- `01_Reference/` — deep internal reference for cosmology, Unity, and bridges
- `00_Project/` — project-level planning and overview material
- `95_Portfolio_Assets/` — exported documents and visual/supporting assets

## Note

The portfolio layer is designed as the public entry point. The reference layer contains deeper internal material and is best read after the overview and case studies.
