# Erit World — Overview

This folder contains cross-era reference files for Erit's world.

Use this folder for information that affects more than one historical era.

## Files To Build

- `Timeline.md` — broad historical sequence from the Celcinus Calendar through Erit's era.
- `Calendar_And_Eras.md` — ACD / Celcinus Calendar, AVD, Year 0 AVD, and the four major eras.
- `Common_Terms.md` — locked names, language drift, house names, civic terms, and recurring terminology.

## Current Era Structure

1. **Founding Era** — how the Regulatus order began.
2. **Fracture Era** — how Ermerus's controlled legacy fractured through his uncontrolled bloodline, allowing Rutare and Ilunis lines to take over.
3. **Devaar Rise Era** — how the rejected Teresta line returned through medicine, preservation, chips, and bodily dependency.
4. **Erit Era** — what the world became by the time Erit, Sera, and Kael enter the story pressure.

## Rule

Overview files should stay practical.

They should help locate information, prevent contradiction, and keep terminology stable.

Do not use overview files for long prose, scene drafts, or detailed character psychology.
