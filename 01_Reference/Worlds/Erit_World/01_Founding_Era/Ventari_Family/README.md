# Ventari Family — Founding Era

This folder contains the founding-era Ventari family files.

These files should focus on the first emotional causes behind the later historical structure.

## Family Files

- `Ermerus_Ventari.md` — last son; controls meaning, public memory, interpretation, and later turns Viriatus's death into Regulatus.
- `Rutare_Ventari.md` — first son; overprotected heir whose insecurity becomes procedure, records, administration, and usefulness.
- `Ilunis_Ventari.md` — second son; overused and maimed at Augunar, later turns humiliation into control, law, force, and punishment.
- `Teresta_Ventari.md` — distances herself after learning the truth, refuses to become a ruling pillar, and becomes the origin point for the rejected line that later returns through Devaar.

## Rule

Do not write these characters as political departments first.

Each file should begin with:

> experience -> emotion -> reason -> behavior

Only after that should the file explain how the behavior becomes atmosphere, history, or later institutional pressure.
